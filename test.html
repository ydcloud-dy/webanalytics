也可以在控制台执行：wa.trackError({message: '测试错误', source: 'js'})

验证步骤

  1. 打开浏览器开发者工具 → Network 标签
  2. 刷新此页面 - 应看到 POST 到 /api/collect 的 pageview 请求
  3. 点击上方的错误测试按钮 - 应看到 POST 到 /api/collect 的 error 请求
  4. 等待几秒（buffer 刷新），然后去仪表盘「错误追踪」页面查看数据
